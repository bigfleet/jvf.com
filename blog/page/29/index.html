Stripping Subversion

by Jim Van Fleet - Feb 10th, 2005 -

or… recursively deleting directories of the same name.

The only tool that I’m comfortable with that could do something like this is Ant. But I thought, hey, you know it’s possible in Unix. It took me about five minutes in Cygwin with man find to come up with this one.

Jim Van Fleet@jimvfxp /cygdrive/c/dev/eclipse-3.1M4/workspace
$ find . -name .svn -exec rm -rf '{}' \;

That did the trick!

Disabling Commons Logging

by Jim Van Fleet - Feb 7th, 2005 -

I’ve developed a suite of test cases for my latest application. In my architecture, I want to make sure that each of the implementations for each of my interfaces passes the test cases as part of the build process. I accomplish this goal using Spring configuration and a little property file maintainance.

As a result, however, following the unit test execution was absolutely impossible. Thanks to this post on the Spring forum, I discovered the key to disabling commons-logging is to set a system property. It was a slam-dunk after that to change a system property using Ant. Fortunately, the JUnit Ant task accepts this property!
