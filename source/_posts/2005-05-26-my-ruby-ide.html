I’ve settled on using jEdit with a plug-in for my Ruby coding. I’ve found that having the core API handy may actually come out higher on the cost/benefit ratio that code completion. I really don’t even miss Ctrl+space.

I am working on one major project in Ruby involving a web application and one minor one involving my MP3 collection.

UPDATE: I have since changed gears and gone with Eclipse and RDT. You can find a fabulous walkthrough for Eclipse and Ruby on Rails at NAPCS.
