Subversion has now cracked the starting line-up, replacing CVS. The cygwin installation (along with the knowledge that Subversion is written to be a client and a server) is enough to get you set up with a local repository after following the simple instructions in the book.

As a devoted Eclipse user, I found it useful to go to the Subclipse update site at http://subclipse.tigris.org/update and use Eclipse’s update facility to install the book and Subclipse itself.

From there, it was no problem at all to get set up! When combined with the upcoming Subversion book from the Pragmatic Programmers, it looks like Subversion has now earned its stripes.
